# SANCHO — Visual Direction

Site one-page ultra premium présentant la direction artistique de la marque SANCHO.

## Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Bebas Neue, DM Serif Display, Syne, Space Mono)
- Images hébergées sur Cloudinary
- Déployé sur Vercel

## Structure

```
sancho-visual-direction/
├── index.html       ← site complet (single-file)
├── 404.html         ← page 404 personnalisée
├── vercel.json      ← headers de sécurité
├── robots.txt
├── sitemap.xml
└── README.md
```

## Sections

1. **Hero** — Fullscreen d'entrée avec logo animé
2. **Le Logotype** — Présentation de la signature graphique
3. **Système de Construction** — Grid, anchors, handles
4. **Recherches & Explorations** — Galerie horizontale scrollable (10 versions)
5. **Version Chrome** — Section fullscreen cinématographique
6. **Applications Textiles** — Collection Originale (4 pièces)
7. **Univers Visuel** — Compositions lifestyle immersives
8. **Section Finale** — CTA téléchargement de la proposition

## Déploiement

### GitHub

```bash
git init
git add .
git commit -m "init: SANCHO Visual Direction"
git remote add origin https://github.com/TON_USERNAME/sancho-visual-direction.git
git push -u origin main
```

### Vercel

1. Connecter le repo GitHub sur [vercel.com](https://vercel.com)
2. Framework Preset → **Other**
3. Build Command → *(vide)*
4. Output Directory → *(vide ou `.`)*
5. Deploy

## Direction Artistique

- **Typographie** : Bebas Neue (display), DM Serif Display (éditorial), Syne (UI), Space Mono (labels)
- **Palette** : Noir absolu `#0A0A0A`, Blanc pur `#F2F2F2`, Chrome métallique
- **Style** : Minimalisme éditorial · Luxe streetwear · Brutalisme premium
- **Animations** : Reveal au scroll · Parallaxe · Cursor custom · Auto-scroll galerie
